# ROSE - Premium Perfume E-Commerce Website

A modern, elegant e-commerce website for premium perfume products built with HTML, CSS, and JavaScript. Features responsive design, smooth animations, and an interactive shopping experience.

## 🌟 Features

- **Modern Homepage** - Eye-catching hero section with smooth animations and scroll-triggered effects
- **Product Shop** - Browse and explore perfume products with 3D model viewer
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Shopping Cart** - Complete checkout functionality
- **About Us** - Company information and brand story
- **Contact/Help** - Customer support page
- **Advanced Animations** - GSAP animations and scroll triggers for engaging user experience
- **Search Functionality** - Built-in search feature to find products easily
- **Mobile Menu** - Hamburger menu for mobile navigation
- **Promotional Banner** - Floating announcement banner for sales and offers

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies or build tools required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/perfume-bottle.git
cd perfume-bottle
```

2. Open the project in your browser:
```bash
# Simply open home.html in your preferred browser
# Or use a local server (recommended)
```

### Running a Local Server

For development, it's recommended to run a local server:

**Using Python 3:**
```bash
python -m http.server 8000
```

**Using Node.js (with http-server):**
```bash
npx http-server
```

Then open `http://localhost:8000` in your browser.
